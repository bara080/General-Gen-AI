General-Gen-AI API

This endpoint is the API gateway for the General-Gen-AI LangGraph application, designed to handle requests for QA, summarization, and streaming.

Available Endpoints (via POST)

  • /api/health (GET) Status Check
  • /api/qa Direct Question Answering
  • /api/summarize Summarization
  • /api/stream Server-Sent Events (SSE)

To interact with the API, please use the /api/ prefix in your application or client.
